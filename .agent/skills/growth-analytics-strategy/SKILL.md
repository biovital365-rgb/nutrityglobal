# Growth & Analytics Strategy Skill

> Focus on user acquisition, retention, and optimization through data-driven decisions.

## Core Rules

| Metric | Focus | Implementation |
|--------|-------|----------------|
| **Retention** | User return rate | Push notifications, Email drip, Personalization |
| **Conversion** | Goal completion | A/B testing, Funnel analysis, CTA optimization |
| **Visibility** | SEO & GEO | Semantic HTML, Schema.org, AI-friendly content |

## Best Practices

1. **Measure Everything:** Use tools like PostHog, Mixpanel, or Google Analytics 4.
2. **Experimentation:** Never roll out a major UI change without an A/B test if possible.
3. **SEO/GEO:** Optimize for Generative Search Engines (Gemini, Perplexity) by providing clear, structured data.
4. **Performance:** CWV (Core Web Vitals) are part of SEO. LCP < 2.5s is mandatory.
5. **Onboarding:** The first 30 seconds define user retention. Optimize for speed and clarity.

## Implementation Patterns

- **Metadata Management:** Dynamic OpenGraph tags for social sharing.
- **Micro-conversions:** Track small steps leading to the main goal (e.g., newsletter signups).
- **Referral Loops:** Incentivize users to invite others.
